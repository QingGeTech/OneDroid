# 📱 OneDroid - System-level Android All-in-one Toolbox

> A powerful Android multi-functional toolkit that integrates various practical tools to help users better manage and understand their Android devices.

English | [中文](README.md)

Official Website: [https://qingge.tech/onedroid/web/](https://qingge.tech/onedroid/web/)


## 🚀 Features

- 📦 **App Management**:
  - App List
  - Basic Info, Signature, Permissions, Components
  - App Operations (Launch, Extract, Uninstall, Modify Internal Files, Decompile)

- 📱 **Device Management**:
  - Device, System, CPU, GPU, Memory, Storage, Screen, Network, Location, Sensors, Camera, Temperature, PROP

- 🔧 **Tool Collection**:
    - Screen Analysis: Long Screenshot (In Progress), Layout Inspector, Screen Recording, Color Picker, Text Recognition
    - Reverse Debugging: Decompile, Logcat (To be Implemented), Terminal (To be Implemented), Mock Location (To be Implemented), WiFi Password (Requires Root)
    - Network Management: Packet Capture (To be Implemented), File Server (To be Implemented)
